description: Fix linting and formatting issues.
---

1. Run `bun run lint --fix` to resolve automatically fixable lint errors.
2. Run `bun run format` (or `prettier --write .`) to ensure consistent styling.
3. Run `git add .` to stage the fixes.
4. Show a summary of changed files to the user.
5. Offer to commit the fixes with a "chore: fix lints and formatting" message.
