# Accessibility First

- **ARIA Attributes**: Use appropriate ARIA roles and attributes for all interactive elements (e.g., `aria-label`, `aria-expanded`).
- **Alt Text**: All `<img>` tags and SVG icons used as images must have descriptive `alt` or `aria-label` text.
- **Semantic HTML**: Prioritize semantic tags (`<main>`, `<header>`, `<footer>`, `<nav>`, `<aside>`, `<button>`, `<a>`) over generic `<div>` and `<span>`.
- **Contrast**: Ensure text colors meet WCAG AA contrast standards.
- **Keyboard Navigation**: Ensure all interactive elements are reachable and usable via keyboard.
