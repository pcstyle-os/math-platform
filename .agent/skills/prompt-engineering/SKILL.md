# prompt-engineering

## Description

Craft effective prompts for Gemini/GPT models within your app.

## Instructions

- Use **Few-Shot Prompting**: Provide 2-3 examples of the desired input/output.
- Use **Chain of Thought**: Ask the model to "think step by step".
- **Structured Output**: Use JSON schemas to ensure the model returns machine-readable data.
- **System Instructions**: Set a strong persona and set of constraints.
- Iteratively test prompts and handle edge cases (e.g., model "refusals").
