# responsive-design

## Description

Build mobile-first, responsive layouts using CSS Grid and Flexbox.

## Instructions

- Start styling for the smallest screen first (Mobile-First).
- Use `grid-template-columns: repeat(auto-fit, minmax(300px, 1fr))` for flexible grids.
- Avoid fixed widths; use percentages, `vw/vh`, or `rem`.
- Use media queries for specific breakpoints (e.g., `@media (min-width: 768px)`).
- Test on different screen sizes using browser dev tools.
- Ensure touch targets are large enough on mobile.
