# performance-profiling

## Description

Analyze and optimize frontend/backend performance bottlenecks.

## Instructions

- Use Chrome DevTools Performance tab to identify long tasks in the frontend.
- Optimize React renders by using `memo`, `useMemo`, and `useCallback`.
- Check backend query performance (e.g., Convex index usage).
- Implement lazy loading for images and heavy components (`next/dynamic`).
- Reduce bundle size by auditing dependencies.
