# data-visualization

## Description

Generate charts and dashboards using D3.js, Chart.js, or Recharts.

## Instructions

- Choose the right chart type (line for trends, bar for comparisons, pie for parts of a whole).
- Ensure visualizations are responsive and accessible (aria-labels, color blind friendly palettes).
- Animate transitions to help users track changes in data.
- Use tooltips and legends to provide context.
- Optimize performance when rendering large datasets (e.g., SVG vs Canvas).
